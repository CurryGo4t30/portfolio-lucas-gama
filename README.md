# Portfólio - Lucas da Silva da Gama

Este é meu portfólio desenvolvido como parte da disciplina de **Análise e Desenvolvimento de Sistemas** na Uninter.

## 🎯 Objetivo
Apresentar minhas informações pessoais, habilidades, formação acadêmica e projetos desenvolvidos, utilizando **HTML5, CSS3 e JavaScript**.

## 🛠️ Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript

## 🚀 Funcionalidades
- Validação de formulário de contato com mensagens estilizadas
- Scroll suave na navegação
- Alternância entre tema claro e escuro
- Botão "voltar ao topo" com rolagem suave

## 📂 Estrutura do projeto
- `index.html` → Página principal
- `style.css` → Estilos do site
- `script.js` → Funcionalidades em JavaScript
